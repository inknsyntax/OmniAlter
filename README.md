# OmniAlter

> The Universal Unit & Currency Converter - Convert Anything, Instantly

**OmniAlter** is a fast, responsive, and comprehensive converter that handles everything from currencies and measurements to gaming XP and fantasy languages. No ads. No backend. Just pure conversion power.

## ✨ Features

### 10 Converter Types
- **💵 Currency** - 180+ currencies with real-time exchange rates (USD, EUR, GBP, JPY, crypto, precious metals, and more)
- **📏 Length** - 15 units (meters, kilometers, miles, feet, inches, nautical miles, and more)
- **⚖️ Weight** - 13 units (kilograms, pounds, ounces, stones, carats, and more)
- **⏱️ Time** - 8 units (seconds, minutes, hours, days, weeks, months, years)
- **🌡️ Temperature** - Celsius, Fahrenheit, Kelvin conversions
- **🧃 Volume** - 14 units (liters, gallons, milliliters, cups, tablespoons, barrels, and more)
- **🚀 Speed** - m/s, km/h, mph, knots, ft/s
- **📐 Area** - m², km², miles², acres, hectares, and more
- **🎮 Gaming XP** - Convert XP to playtime across 35+ games with skill level multipliers (Casual, Normal, Hardcore, Speedrunner)
- **🗣️ Language** - 20+ real languages + 4 fantasy languages (Elvish, Klingon, Na'vi, Pirate English)

### Design & UX
- 🎨 **13 Color Themes** - Choose from Clean, Currency, Gaming, Historical, Fantasy, and more
- 📱 **Fully Mobile Responsive** - Optimized for phones, tablets, and desktops
- ⚡ **Instant Conversions** - Real-time conversion as you type
- 🔄 **Theme Toggle** - Compact theme selector on mobile
- 🚀 **Zero Dependencies** - Pure HTML, CSS, and JavaScript
- 🎯 **Professional UI** - Clean, modern design with proper typography and spacing

## 💡 Usage

### Currency Conversion
1. Select "Currency" tab
2. Enter amount in the "From" field
3. Choose currencies and watch real-time conversion

### Gaming XP to Time
1. Select "Gaming" tab
2. Pick your game
3. Enter XP amount
4. Select skill difficulty
5. See estimated playtime

### Language Translation
1. Select "Language" tab
2. Enter text or phrase
3. Choose target language
4. Click "Translate"
5. See translated result

### Theme Customization
- **Desktop**: Click any theme button in the header
- **Mobile**: Tap "Themes" button to reveal all theme options

## 🛠️ Technical Details

**Stack:**
- HTML5 for semantic structure
- CSS3 with custom properties for theming
- Vanilla JavaScript (no frameworks or libraries)

**Architecture:**
- Base-unit conversion pattern for all measurements
- Dynamic DOM population from data objects
- CSS custom properties for instant theme switching
- Responsive design with mobile-first approach

**File Structure:**
```
├── index.html      (UI structure, styles, theme selector)
├── script.js       (All conversion logic & data)
├── style.css       (Reserved for future modularization)
└── README.md       (This file)
```
